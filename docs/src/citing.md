# Citing

If you wish to cite \<package\>, please use:

## Original paper

Main paper.

```latex
@article{paper2016,
    author       = {authors},
    title        = {title of the paper},
    journal      = {journal name},
    month        = {June},
    year         = {2016},
    volume       = {169},
    number       = {3},
    pages        = {1042-1068},
    url          = {url to the paper},
}
```
